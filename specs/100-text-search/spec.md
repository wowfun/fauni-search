# 100 文本搜索 (Text Search)

定义 FauniSearch 的文本搜索能力域，明确文本查询如何在单个库上下文中作用于已进入可搜索状态的视觉内容，并返回可消费的搜索结果。

## 关键术语 (Terminology)

- 文本搜索（Text Search）
- 可搜索状态（Searchable State）
- 文本搜索结果（Text Search Result）
- 未就绪状态（Not Ready State）

## 范围

- 单个库上下文中的文本查询能力
- 文本搜索的长期结果对象边界
- 内容进入可搜索状态后的搜索可用性、未就绪与失败反馈语义
- 文本搜索与上游摄取、索引、搜索接口契约之间的复用边界

范围外：
- 具体请求 / 响应字段、错误载荷、游标编码与资源引用对象形状
- 具体模型、索引算法、融合排序与检索后端实现
- 具体接入入口，例如手动导入、来源根扫描或其他控制面形状
- 对象详情、邻近上下文与展开接口的具体公开编码
- 当前阶段实现范围、阶段验收步骤与 UI 形态

## 设计原则

- 单库上下文（Single-Library Scope）：文本搜索始终作用于单个目标库，不发生跨库隐式搜索
- 可搜索先于查询（Searchability Before Query）：文本搜索只应依赖已启用且已进入可搜索状态的索引线
- 视觉对象优先（Visual Unit First）：文本搜索默认返回视觉单元结果，而不是先按源内容聚合
- 显式状态（Explicit Searchability State）：未就绪、失败与无结果必须保持可区分，而不是互相伪装
- 事实源复用（Fact-Source Reuse）：文本搜索复用上游摄取、索引、搜索与接口契约专题，不在本专题重复定义其稳定事实

## 能力边界

- 文本搜索只接受文本查询输入
- 文本搜索始终作用于单个目标库
- 文本搜索长期上允许以下结果对象参与命中：
  - `document_page`
  - `image`
  - `video_segment`
- 文本搜索只能依赖目标库内已启用且已进入可搜索状态的索引线
- 文本搜索结果的对象详情、邻近上下文与展开路径继续复用上游搜索与接口契约专题，不在本专题单独定义其公开形状

## 可搜索状态

- 库内容只有在经过正式接入并完成相应索引线的构建后，才进入该索引线上的可搜索状态
- 可搜索状态以索引线为作用域；同一库的不同索引线可以处于不同可搜索状态
- 目标索引线已启用但尚未持有可服务搜索的 active index 时，应视为未就绪，而不是“已有结果但暂时为空”
- 正式接入、索引构建或索引激活失败导致内容无法进入可搜索状态时，系统必须对外表达明确失败或不可搜索反馈
- 仅有库、源内容或已接入记录存在，并不自动意味着文本搜索已经可用

## 功能需求

### 查询

- `FR-1` 系统必须允许用户在单个目标库上下文中发起文本查询。
- `FR-2` 文本查询只能作用于目标库内已启用且已进入可搜索状态的索引线，不得发生跨库隐式搜索。
- `FR-3` 当文本查询超出支持范围、命中未启用索引线或命中未就绪索引线时，系统必须返回显式拒绝或显式未就绪语义，而不是静默降级、忽略或返回空结果。

### 结果

- `FR-4` 文本搜索必须默认返回视觉单元粒度的结果，而不是先按源内容聚合后返回。
- `FR-5` 文本搜索长期上必须允许 `document_page`、`image` 与 `video_segment` 作为可命中的结果对象。
- `FR-6` 文本搜索结果必须能够以稳定、可消费的最小结果卡片形式被展示，并支持后续通过上游搜索 / 详情能力继续展开。

### 可搜索状态

- `FR-7` 内容只有在正式接入与相关索引完成后，才能进入可被文本搜索命中的状态。
- `FR-8` 当目标索引线尚未就绪时，系统必须向用户表达“尚不可搜索”或等价未就绪状态，而不是静默返回空结果。
- `FR-9` 当正式接入、索引或激活失败导致文本搜索不可用时，系统必须向用户表达明确失败或不可搜索反馈，而不是仅表现为无结果。

## 验收标准

- `AC-1` 当目标库中的相关内容已进入可搜索状态后，文本查询能够命中 `document_page` 结果。
- `AC-2` 当目标库中的相关内容已进入可搜索状态后，文本查询能够命中 `image` 结果。
- `AC-3` 当目标库中的相关内容已进入可搜索状态后，文本查询能够命中 `video_segment` 结果。
- `AC-4` 当查询同时命中多类视觉对象时，同一结果集中允许混排返回这些结果。
- `AC-5` 当请求涉及已启用但尚未就绪的索引线时，系统返回明确未就绪反馈，而不是静默返回空结果。
- `AC-6` 当正式接入、索引或激活失败导致文本搜索不可用时，系统返回明确失败或不可搜索反馈，而不是只表现为无结果。
- `AC-7` 文本查询只作用于单个目标库，不发生跨库隐式搜索。
