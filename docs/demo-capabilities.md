# 对外演示能力清单

本文用于收敛当前阶段最适合对外展示的 FauniSearch 能力，面向需要快速理解“这个项目现在能演示什么”的场景，帮助演示者在有限时间内展示最有感知的能力。

## 适用场景

- 面向业务、产品或潜在用户做 10 分钟左右的能力展示
- 需要从现有实现中挑出最容易被理解的亮点，而不是完整讲解架构
- 需要在现场前快速确认“哪些能力适合讲，哪些能力更适合留到问答”

## 当前最适合展示的核心能力

- 以文本作为查询 (`Text` 模式)
- 以图片作为查询 (`Image` 模式)
  - 支持上传单张查询图片
  - 支持把库内 `image` 或 `document_page` 结果对象直接复用为新的 query image
- 以视频作为查询 (`Video` 模式)
  - 支持上传本地查询视频
  - 支持整段视频查询，也支持指定时间范围的视频片段查询
  - 支持把库内 `video_segment` 直接复用为新的查询视频片段
- 以文档作为查询 (`Document` 模式)
  - 支持上传 PDF 作为查询文档
  - 支持整份文档查询，也支持 `start_page/end_page` 页范围查询
  - 支持把库内 `document_page` 结果对象直接复用为新的查询文档
